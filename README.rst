lizard-3ditask
==========================================

An app for running 3Di scenarios. It currently runs under Windows.


Prerequisites
-------------

- 3Di 0.1.1.224:225M or newer
- Psycopg2
- Setuptools
- Numpy 1.6.2 (tested)
- Tested with Python 2.6


Development installation
------------------------

Checkout, install the prerequisites and run:

    > python bootstrap.py
    > bin\buildout
